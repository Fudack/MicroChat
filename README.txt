# API de Administración de Usuarios

Esta API es responsable de administrar los usuarios en una aplicación.

## Funcionalidades

- Registro de nuevos usuarios
- Inicio de sesión de usuarios existentes
- Actualización de información de usuarios
- Eliminación de usuarios

## Endpoints

- `POST /usuarios`: Crea un nuevo usuario.
- `POST /usuarios/login`: Inicia sesión de un usuario existente.
- `PUT /usuarios/:id`: Actualiza la información de un usuario.
- `DELETE /usuarios/:id`: Elimina un usuario.

## Uso

1. Clona este repositorio.
2. Instala las dependencias con `npm install`.
3. Configura las variables de entorno en un archivo `.env`.
4. Ejecuta la API con `npm start`.
